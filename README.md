# Robin Wood Front

Next.js TypeScript project with route groups architecture.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/(auth)/` - Authentication routes (login, register)
- `app/(public)/` - Public routes (about, contact)
- `app/` - Root layout and home page

## Routes

- `/` - Home page
- `/login` - Login page
- `/register` - Register page
- `/about` - About page
- `/contact` - Contact page
